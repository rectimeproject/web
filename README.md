# RecTime - Next.js Audio Recording Application

A modern, professional audio recording web application built with Next.js 14, TypeScript, and TailwindCSS. Features high-quality Opus codec recording with a beautiful dark/light theme UI.

## 🚀 Features

- **Modern UI/UX**: Built with Next.js 14 App Router and TailwindCSS
- **Dark/Light Theme**: Automatic theme detection with manual toggle
- **High-Quality Audio**: Opus codec support for superior audio compression
- **Real-time Visualization**: Live audio waveform visualization during recording and playback
- **Persistent Storage**: IndexedDB-based recording storage
- **Responsive Design**: Works seamlessly on desktop and mobile devices
- **Type-Safe**: Full TypeScript support

## 🛠️ Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Styling**: TailwindCSS
- **Language**: TypeScript
- **Audio**: Web Audio API, Opus Codec, standardized-audio-context
- **Storage**: IndexedDB (idb-javascript)
- **Theme**: next-themes

## 📦 Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm start` - Start production server
- `npm run lint` - Run ESLint

## 🔧 Installation

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
